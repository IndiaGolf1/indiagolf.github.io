Types of Computer Viruses

Not all malware behaves the same way.

Common Categories (Unordered List)

  • File Infectors — attach to executable files.
  • Boot Sector Viruses — infect the master boot record.
  • Macro Viruses — embedded in documents (Word, Excel).
  • Polymorphic Viruses — change their code to evade detection.
  • Resident Viruses — install themselves in system memory.

A Typical Infection Lifecycle (Ordered List)

  1. User opens an infected file or link.
  2. Malicious code executes silently.
  3. Virus replicates and attaches to other files.
  4. Payload activates (data theft, encryption).
  5. Virus spreads across the network.
